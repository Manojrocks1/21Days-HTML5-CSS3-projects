# Day 1: Personal Website

This is a simple personal website created as part of a 21-day HTML and CSS learning plan.

## 📄 Project Overview

The project includes:
- A personal bio section
- Contact information (email and phone)
- A circular profile photo (using a placeholder)
- Clean layout and simple CSS styling
- Responsive design for mobile compatibility
- A footer section

## 📁 Files

- `index.html` — The main HTML structure of the webpage
- `personal.css` — Styling file for layout, spacing, and design
- `profile.jpg` — (Optional) Profile picture; using a placeholder by default

## 🚀 How to Run

1. Download or clone this repository.
2. Open `index.html` in any modern web browser.

## 💡 Preview

The site displays a centered card with your name, profile image, short intro, and contact details.

## ✅ Skills Practiced

- HTML structure and elements
- Linking external CSS
- Styling text, containers, images
- Responsive design basics
- Semantic HTML and accessibility considerations
